# Econova 

Este repositorio contiene el proyecto **Econova**, desarrollado para el **Trabajo Intermodular de Grado Medio en Sistemas Microinformáticos y Redes (SMR)**. 

## 🚀 Estado del Proyecto (Actualizado a: 28/04/2026)

Actualmente, el repositorio se encuentra en fase de despliegue progresivo:

- [x] **Landing Page:** Finalizada y publicada. Una página de aterrizaje de alto impacto visual inspirada en la estética "Punk-Deco" de *Persona 5*, diseñada para captación y auditoría Eco-Index.
- [ ] **Configuración de Servidores:** (Pendiente de subir) Documentación y scripts de configuración de la infraestructura local (Debian/Windows Server, DNS, DHCP, etc.).
- [ ] **Econova Web App:** (En desarrollo) Portal interno de gestión de inventario y marketplace de validación técnica.

## 🛠️ Tecnologías Usadas en la Landing

![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white)

* **CSS3 (Flexbox):** Maquetación responsiva y diseño asimétrico sin frameworks.
* **JavaScript:** Validación de formularios y efectos dinámicos de feedback visual.

## 🎨 Concepto Visual
El diseño rompe con la estética tradicional corporativa, utilizando ángulos agresivos, contrastes en **Naranja y Azul Eléctrico**, y una estructura de "recortes" inspirada en **Persona 5 Royal**.
